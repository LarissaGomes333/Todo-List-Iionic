
# To do daily

O to do daily é um sistema de lista de tarefas, onde você pode organizar suas tarefas como quiser e ir dando check conforme as completa.

- Ionic
- Angular
- TypeScript
- Firebase
## Funcionalidades

- Cadastro e Login
- Ler, Criar, Alterar e Deletar Notas
